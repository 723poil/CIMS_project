 + 전체 사용자에게 전달 시
 
   - 사용자의 앱에서는 일정 시간마다 데이터베이스를 검사
   - read : False 일 경우 알림 게시
   - read : True 일 경우 무시

   - 특정 사용자의 데이터베이스 안에 알림 관련 내용 추가

   - read : True 일 경우 무시
